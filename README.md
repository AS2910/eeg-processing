# eeg-processing
This contains a generic EEG processing pipeline inspired by the HAPPE pipeline, in Matlab using EEGLab.<br>
The folder structure is stated below:<br>
Main_Folder<br>
  |Subject1<br>
    |EEG_File1<br>
    |EEG_File2<br>
    |EEG_File3<br>
    |EEG_File4<br>
    |....<br>
  |Subject2<br>
    |EEG_File1<br>
    |EEG_File2<br>
    |EEG_File3<br>
    |EEG_File4<br>
    |....<br>
  |...<br>
  <br>
